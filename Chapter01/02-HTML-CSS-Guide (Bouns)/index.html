HTML & CSS Quick Reference Guide

This guide provides a quick overview of the most common HTML tags and CSS properties, along with examples to help you understand how they work.

Common HTML Tags

  • <h1> - <h6> → Headings
  • <p> → Paragraph
  • <a> → Link
  • <img> → Image
  • <ul> / <ol> → Lists
  • <li> → List Item
  • <div> → Container
  • <span> → Inline text
  • <code> → Inline code
  • <pre> → Preformatted code

Example:


<h1>Welcome to My Page</h1>
<p>This is a paragraph.</p>
<a href="#">Click Here</a>

Common CSS Properties

  • color → Text color
  • background-color → Background color
  • padding → Inner spacing
  • margin → Outer spacing
  • border → Border styling
  • font-size → Text size
  • font-family → Font style
  • text-align → Text alignment
  • border-radius → Rounded corners

Example:


.box {
    background-color: #222;
    color: #fff;
    padding: 20px;
    border-radius: 10px;
}
